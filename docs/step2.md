## 기능 요구 사항
사다리 게임에 참여하는 사람에 이름을 최대5글자까지 부여할 수 있다. 사다리를 출력할 때 사람 이름도 같이 출력한다.
사람 이름은 쉼표(,)를 기준으로 구분한다.
사람 이름을 5자 기준으로 출력하기 때문에 사다리 폭도 넓어져야 한다.
사다리 타기가 정상적으로 동작하려면 라인이 겹치지 않도록 해야 한다.
|-----|-----| 모양과 같이 가로 라인이 겹치는 경우 어느 방향으로 이동할지 결정할 수 없다.

## 힌트
2차원 배열을 ArrayList, Generic을 적용해 구현하면 ArrayList<ArrayList<Boolean>>와 같이 이해하기 어려운 코드가 추가된다.
사다리 게임에서 한 라인의 좌표 값을 가지는 객체를 추가해 구현해 본다.

## 기능 구현 목록
- [x] 사다리 도메인을 구현한다.
  - [x] 사다리 높이는 최소 1 이상이다.
  - [x] 사다리 게임에 참여하는 사람은 최소 2명 이상이다.
  - [x] 가로 라인을 생성하는 전략을 구현한다.
    - [x] 가로 라인은 랜덤으로 생성한다.
    - [x] 가로 라인은 겹치지 않도록 생성한다.
- [ ] 사다리 게임에 참여하는 사람의 이름을 입력받는다.
  - [ ] 사다리 게임에 참여하는 사람의 이름은 쉼표(,)로 구분한다.
  - [x] 사다리 게임에 참여하는 사람의 이름은 최대 5글자까지 가능하다.
- [ ] 사다리 높이를 입력받는다.
- [ ] 실행 결과를 출력한다.
  - [ ] 사다리 게임에 참여하는 사람의 이름을 5글자 기준으로 출력한다.
    - [ ] 참여자 이름 길이에 따라 이름 배치가 달라진다.
  - [ ] 사다리 높이, 가로 라인에 따라 사다리를 구현한다.
