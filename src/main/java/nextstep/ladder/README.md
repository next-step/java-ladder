# 기능 요구 사항

## step-2: 사다리(생성)

- [ ] 사다리 게임에 필요한 값을 입력받는다.
    - [x] 참여할 사람을 쉼표로 구분해 최대 5글자까지 입력받는다.
        - [x] 참여자는 최소 2명이어야 한다.
    - [x] 사다리 최대 높이를 입력받는다.
- [ ] 사다리를 생성한다.
    - [ ] 가로로 발판 라인을 생성한다.
        - [x] 인원 수 만큼 가로 라인에 칸을 만든다.
        - [x] 해당 칸에 발판을 놓을지 여부는 랜덤하게 정한다.
        - [x] 발판은 가로로 겹치지 않는다.
    - [ ] 가로 라인을 높이만큼 반복해서 생성한다.
- [ ] 결과를 출력한다.
