# 기능명세
## 입력
- [ ] 사다리 게임에 참여하는 사람이름을 입력받는다
  - [ ] 이름은 최대 5글자이다
  - [ ] 최소 1명 이상의 이름을 입력받아야 한다.
  - [ ] 이름의 구분 기분은 (,) 이다. 
- [ ] 사다리의 높이를 입력받는다.

## 출력
- [ ] 사다리 폭은 5개의 (-)로 한다.
- [ ] 참여자 이름도 함께 출력한다.

## 도메인
### 사다리
- [ ] 사다리의 가로 라인이 겹칠 수 없다.
- [ ] 사다리의 세로줄 수는 참가자 수와 같다.
- [ ] 사다리의 높이는 입력받은 높이만큼 생성된다.

### 라인
- [ ] 라인은 참가자 수 -1 으로 Point를 가진다.
- [x] 마지막 라인의 hasPoint는 false여야 한다.
- [ ] 두 개 이상의 가로줄이 연속으로 연결되면 안 된다.
