## 2 단계 - 사다리(생성)

### 기능 요구사항

#### 입력

* [ ] 사다리 게임에 참여하는 사람의 이름은 최대 5자까지 허용한다.
* [ ] 사다리를 출력할 때 사람의 이름도 같이 출력된다.
* [ ] 사람의 이름은 ,를 기준으로 구분된다.
* [ ] 최대 사다리 높이를 결정할 수 있다.

#### 출력

* [ ] 주어진 사람의 이름에 맞춰 사다리 폭도 넓어져야 한다.
* [ ] 사다리끼리 겹치지 않는다.
* [ ] 사다리 모양은 |---|---| 같이 정한다.
* [ ] 세로축 사다리는 사람의 이름 끝점에 맞춰서 출력한다.

#### 사다리

* [ ] 사람의 수 만큼 세로축 사다리를 생성한다.
* [ ] 세로축 사다리는 최대 사다리 높이를 넘을 수 없다.
* [ ] 가로축 사다리의 크기 = (가장 긴 이름의 길이) + 2이다.
