# 사다리 게임

## 2단계

### 기능 목록

- 입력
  - 참여할 `사람의 이름`
    - 사람의 이름은 최소 1자, 최대 5자까지 입력할 수 있다.
    - 사람의 이름은 쉼표(`,`)를 기준으로 구분한다.
  - `사다리 높이`
    - 최소 1 이상의 숫자를 입력할 수 있다.
- 사다리
  - `|-----|-----|` 모양과 같이 가로 라인이 겹치지 않아야 한다.
- 출력
  - 참여한 `사람의 이름`
  - `사다리`
    - 높이 : `|`
    - 폭 : `-----`, 사람 이름의 길이인 최대 5자 기준
