# 사다리 게임
### 사다리(생성)
- [x] 참여자 이름은 최대 5글자이다.
- [x] 참여자 이름은 쉼표를 기준으로 구분되어 입력된다.
- [x] 참여자는 두 명 이상이다.
- [x] 참여자 이름은 중복 될 수 없다. 
- [x] 사다리의 한 라인은 (참여자의 수 - 1) 만큼의 포인트를 갖는다. 
- [x] 사다리의 한 라인은 연속으로 연결된 포인트를 가질 수 없다. (ex: |-----|-----|)
- [x] 참여자 수에 따라 포인트를 랜덤으로 생성 할 수 있다. 
- [x] 사다리는 입력 받은 높이 만큼의 라인을 갖는다. 
- [ ] 사다리를 출력 할 수 있다.
