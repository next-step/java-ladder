# 사다리 게임

## 진행 방법

* 사다리 게임 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 코드 리뷰 요청을 한다.
* 코드 리뷰 피드백에 대한 개선 작업을 하고 다시 PUSH한다.
* 모든 피드백을 완료하면 다음 단계를 도전하고 앞의 과정을 반복한다.

## 온라인 코드 리뷰 과정

* [텍스트와 이미지로 살펴보는 온라인 코드 리뷰 과정](https://github.com/nextstep-step/nextstep-docs/tree/master/codereview)

### 기능 요구 사항

- [x] 사다리 게임에 참여하는 사람의 이름을 입력받는다.
    - [x] 최대 5글자까지 부여할 수 있다.
    - [x] 쉼표를 기준으로 구분한다.
- [x] 사다리 높이를 입력받는다.
- [x] 사다리의 가로 라인이 겹치지 않게 생성한다.
    - [x] 한 라인의 좌표 값을 가지는 객체를 추가해 구현해 본다.
- [x] 사다리와 사람 이름을 같이 출력한다.
- [x] 실행 결과를 입력받는다.
    - [x] 쉼표를 기준으로 구분한다.
    - [x] 참여자 수보다 결과가 많으면 예외를 반환한다.
- [x] 사다리 게임을 진행한다.
    - [x] 모든 참가자에 위치 정보를 추가한다.
    - [x] 현재 위치에 사다리가 있는지 판단한다.
        - [x] 사다리가 없으면 한 칸 이전에 사다리가 있는지 확인한다.
    - [x] 사다리를 타고 위치를 이동한다.
        - [x] 현 위치에 사다리가 있으면 위치값이 증가한다.
        - [x] 존재하지 않으면 좌측 사다리 여부에 따라 위치값이 감소하거나 그대로이다.
- [x] 결과를 보고 싶은 사람을 입력받는다.
    - [x] 'all'과 입력받은 사람 외에는 예외를 발생시킨다.
- [x] 사다리 실행 결과를 출력한다.
    - [x] 개인별 이름을 입력하면 개인별 결과를 출력한다.
    - [x] "all"은 전체 참여자의 실행 결과를 출력한다.

### 제약사항

- 3개 이상의 인스턴스를 가진 클래스를 갖지 않는다.
