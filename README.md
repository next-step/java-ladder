# 기능 구현 목록

### 입력
- [x] 사람의 이름을 입력한다.
  - [x] 입력한 값을 반환한다.
  - [x] 쉼표로 사람 이름을 구분한다.
  - [x] 최대 5글자까지 부여할 수 있다.
- [ ] 최대 사다리 높이를 입력한다.

### 출력
- [ ] 사다리를 출력한다.
  - [ ] 사람 이름을 사다리 위에 같이 출력한다.
  - [ ] 사다리의 폭은 '-' 5개 길이로 설정한다.

### 도메인
- [x] 사다리의 라인을 설정한다.
  - [x] 사다리를 생성한다.
  - [x] 가로라인이 겹치지 않아야 한다.
  - [x] 가로라인의 맨 끝은 무조건 부분 라인이 없어야 한다.
