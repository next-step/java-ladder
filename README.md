## STEP-02

### 입력

* [X] 사람의 이름은 최대 5글자이다
* [X] 쉼표(,) 를 기준으로 이름을 구분한다

### 출력

* [X] 5자 기준으로 사용자 이름을 출력한다
* [X] 사다리를 출력 한다

### 기능

* [X] 사다리 타기가 정상적으로 동작하기 위해 라인이 겹치지 않도록 한다
* [X] 높이에 맞는 사다리 생성한다
