# 사다리 게임
## 진행 방법
* 사다리 게임 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 코드 리뷰 요청을 한다.
* 코드 리뷰 피드백에 대한 개선 작업을 하고 다시 PUSH한다.
* 모든 피드백을 완료하면 다음 단계를 도전하고 앞의 과정을 반복한다.

## 온라인 코드 리뷰 과정
* [텍스트와 이미지로 살펴보는 온라인 코드 리뷰 과정](https://github.com/nextstep-step/nextstep-docs/tree/master/codereview)

## 기능 목록
- [x] 입력(InputView)
    - [x] 참여자 입력
      - [x] 이름 쉼표 구분 
      - [x] 이름 5자 초과시 예외 처리
      - [x] 실행 결과 입력(꽝,5000,...)
    - [x] 사다리 높이 입력
      - [x] 높이 1미만 시 예외처리
- [x] 라인(line)
  - [x] 연결 상태 확인
  - [x] 연결 (연결 되있지 않을 시)
  - [x] 연결 상태에 따른 위치 이동
- [x] 특정 높이에서의 라인들(lines)
  - [x] 라인 생성
    - [x] 연속 생성 불가 
- [x] 사다리(Ladders)
  - [x] 각 높이 라인들 생성
- [x] 결과 (Result)
  - 입력 결과를 감싸는 클래스
- [x] 사다리 결과 (LadderResult)
  - [x] 참여자 당첨 결과 여부
- [x] 출력 (ResultView)
  - [x] 사다리 상태 출력
  - [x] 결과를 보고 싶은 사람 출력
